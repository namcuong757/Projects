--Simple Library Management--

==Description==

This is Library Management program written in C++ useing text file as a "data base".
When you first start the program.The library will be empty. However, after interact with it. Program will save all of interactive data pernamently.

Note
**
You only have the option to return item in the first menu.
More access provided after you log into the admin menu with default password.
Default password: password
